# brtk-minimal
BRowser ToolKit minimal compornent
## install by mrbgems
- add conf.gem line to `build_config.rb` or brtk

```ruby
MRuby::Build.new do |conf|

    # ... (snip) ...

    conf.gem :github => 'wanabe/brtk-minimal'
end
```
